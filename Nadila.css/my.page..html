Selamat Datang Di Portopolio Saya

Portofolio saya di Roblox berisi pengalaman saya main dan jelajahin berbagai map, terutama map favorit saya yaitu Fist It. Di game itu saya sering latihan buat naikin level, upgrade kekuatan, dan tanding sama player lain. Dari situ saya belajar gimana cara bersaing tapi tetap seru, serta ngerti sistem progres dalam sebuah game.

Selain itu, lewat Roblox saya juga bisa nunjukin kreativitas, karena di sana bukan cuma main tapi juga bisa belajar bikin dunia atau game sendiri. Pengalaman main Roblox bikin saya lebih paham tentang desain map, cara interaksi antar pemain, dan gimana sebuah game bisa bikin orang betah main lama. Buat saya, portofolio ini jadi bukti kalau hobi main Roblox juga bisa bermanfaat dan nambah pengalaman di dunia game.
